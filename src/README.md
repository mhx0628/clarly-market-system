# 澄迈市场管理系统微信小程序

## 项目简介

本项目旨在为澄迈菜篮子市场提供一个智能化的管理平台，整合商户自助服务、管理员协作办公以及领导监控功能，提升市场运营效率。

## 目录结构

- `src/`: 源代码目录
  - `api/`: 后端API接口
  - `app/`: 小程序端代码
  - `assets/`: 静态资源
  - `components/`: 可复用组件
  - `config/`: 配置文件
  - `db/`: 数据库相关
  - `lib/`: 第三方库
  - `pages/`: 页面
  - `styles/`: 样式文件
  - `utils/`: 工具函数
- `docs/`: 文档目录
- `tests/`: 测试目录
